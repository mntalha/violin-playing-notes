# 🎼 How to Read Music for Violin

---

## 1. The Staff — Your Music Grid

Music is written on 5 lines with 4 spaces between them:

```
  Line 5:  ─────────────────────────────  F
  Space 4:                                E
  Line 4:  ─────────────────────────────  D
  Space 3:                                C
  Line 3:  ─────────────────────────────  B
  Space 2:                                A
  Line 2:  ─────────────────────────────  G
  Space 1:                                F
  Line 1:  ─────────────────────────────  E
```

### Memory Tricks:

```
  LINES  (bottom to top):  E  G  B  D  F
         "Every Good Boy Does Fine"

  SPACES (bottom to top):  F  A  C  E
         Spells "FACE"
```

## 2. Where Each Note Sits on the Staff → Your Violin

```
  Staff Position          Note    Code    String    Finger
  ──────────────          ────    ────    ──────    ──────
  above (ledger line)      B      E4     E str     4th
  above line 5             A      E3     E str     3rd
                           G      E2     E str     2nd
  ─────── Line 5 ───────   F      E1     E str     1st
  ○ in space 4             E      E0     E str     OPEN (or A4)
  ─────── Line 4 ───────   D      A3     A str     3rd
  ○ in space 3             C      A2     A str     2nd
  ─────── Line 3 ───────   B      A1     A str     1st
  ○ in space 2             A      A0     A str     OPEN (or D4)
  ─────── Line 2 ───────   G      D3     D str     3rd
  ○ in space 1             F      D2     D str     2nd
  ─────── Line 1 ───────   E      D1     D str     1st
  just below staff         D      D0     D str     OPEN (or G4)
  ● on ledger line         C      G3     G str     3rd
  below                    B      G2     G str     2nd
  ── on ledger line        A      G1     G str     1st
                           G      G0     G str     OPEN
```

### Visual Map: Staff to Fingerboard

```
  STAFF                           FINGERBOARD
  
  ──F── (line 5)          ───────[1]─── E string  = E1
    E   (space 4)         open───────── E string  = E0
  ──D── (line 4)          ───────[3]─── A string  = A3
    C   (space 3)         ───────[2]─── A string  = A2
  ──B── (line 3)          ───────[1]─── A string  = A1
    A   (space 2)         open───────── A string  = A0
  ──G── (line 2)          ───────[3]─── D string  = D3
    F   (space 1)         ───────[2]─── D string  = D2
  ──E── (line 1)          ───────[1]─── D string  = D1
                          open───────── D string  = D0
  (ledger lines below)    fingers on G string
```

## 3. Note Durations — How Long to Hold

```
  Symbol    Name            Beats    Looks Like
  ──────    ────            ─────    ──────────
  𝅝         Whole note       4       ○          (empty, no stem)
  𝅗𝅥         Half note        2       𝅗𝅥          (empty + stem)
  ♩         Quarter note     1       ♩          (filled + stem)
  ♪         Eighth note      ½       ♪          (filled + stem + flag)
  ♬         Sixteenth note   ¼       ♬          (filled + stem + 2 flags)
```

### Counting them:

```
  4/4 time = 4 beats per measure

  |  𝅝                    |     ← 1 whole note fills the bar
  |  1    2    3    4     |

  |  𝅗𝅥         𝅗𝅥         |     ← 2 half notes fill the bar
  |  1    2    3    4     |

  |  ♩    ♩    ♩    ♩    |     ← 4 quarter notes fill the bar
  |  1    2    3    4     |

  |  ♪♪   ♪♪   ♪♪   ♪♪  |     ← 8 eighth notes fill the bar
  |  1 +  2 +  3 +  4 + |
```

## 4. Rests — When to Stay Silent

```
  Symbol    Name            Beats of Silence
  ──────    ────            ────────────────
  𝄻         Whole rest       4  (hangs from line 4)
  𝄼         Half rest        2  (sits on line 3)
  𝄽         Quarter rest     1  (squiggly zig-zag)
  𝄾         Eighth rest      ½  (like a 7)
```

## 5. Time Signatures

The two numbers at the start of a piece:

```
  4    ← 4 beats per measure
  ─
  4    ← quarter note gets 1 beat

  3    ← 3 beats per measure (waltz feel)
  ─
  4    ← quarter note gets 1 beat

  2    ← 2 beats per measure (march feel)
  ─
  4    ← quarter note gets 1 beat
```

## 6. Key Signatures — Which Notes Are Sharp or Flat

```
  Key Signature        Sharps/Flats         Common in Violin?
  ─────────────        ────────────         ─────────────────
  C Major / A minor    none                 ✅ Very common
  G Major / E minor    F#                   ✅ Very common
  D Major / B minor    F#, C#               ✅ Most common!
  A Major / F# minor   F#, C#, G#           ✅ Common
  F Major / D minor    B♭                   ✅ Common
  B♭ Major / G minor   B♭, E♭              Sometimes
```

### What it means:
```
  If you see ONE sharp (F#) at the start:
  → You're in G Major (or E minor)
  → EVERY F in the piece is played as F# 
  → (high 2nd finger on D string, high 1st finger on E string)
```

## 7. Essential Symbols You'll See

```
  Symbol     Meaning
  ──────     ───────
  #          Sharp — raise note by half step
  ♭          Flat — lower note by half step
  ♮          Natural — cancel sharp/flat
  .          Dot after note — adds half the note's value
             (dotted half note = 3 beats)
  ‿          Tie — connect two notes, hold for combined length
  ⌢          Slur — play notes in one bow stroke
  >          Accent — play louder/stronger
  p          Piano — play soft
  f          Forte — play loud
  mp         Mezzo piano — medium soft
  mf         Mezzo forte — medium loud
  <          Crescendo — gradually louder
  >          Decrescendo — gradually softer
  ▼          Down bow (frog to tip)
  ▲          Up bow (tip to frog)
  rit.       Ritardando — slow down
  ×          Pizzicato — pluck with finger
```

## 8. Reading Practice — Put It Together

### Exercise: Name these notes (answers below)

```
  ──────────────────────────────────
        ●                    ●
  ──────────●──────────────────────
                   ●
  ────────────────────●────────────
              ●
  ──────●──────────────────────────
  
  Position: 1    2    3    4    5    6
```

**Answers (read left to right):**

```
  Position:  1      2      3      4      5      6
  Note:      G      A      B      C      D      E
  Staff:     line2  sp2    line3  sp3    line4  sp4
  Code:      D3     A0     A1     A2     A3     E0
  String:    D str  A str  A str  A str  A str  E str
  Finger:    3rd    open   1st    2nd    3rd    open
```

## 9. Ledger Lines

Notes above or below the staff use small extra lines:

```
  ── ● ──   B  (above staff, E string 4th finger)
  ── ● ──   A  (above staff, E string 3rd finger)
     ●      G  (just above line 5, E string 2nd finger)
  ═════════ F  (line 5)
  
  
  ═════════ E  (line 1)
     ●      D  (just below staff)
  ── ● ──   C  (ledger line, G string 3rd finger)
     ●      B  (below that)
  ── ● ──   A  (ledger line, G string 1st finger)
     ●      G  (open G string)
```
