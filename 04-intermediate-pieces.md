# 🎶 Intermediate Pieces & Techniques (Level 2)

*You can now use all 4 fingers, cross strings, and play basic scales.*
*Time to add: slurs, dynamics, string crossings, and 4th finger.*

**Code format:** `A1` = A string, 1st finger. `E0` = E string, open. `D3` = D string, 3rd finger.

---

## New Techniques at This Level

### 1. Slurs (Multiple Notes in One Bow)

```
  Without slur:          With slur:
  ♩  ♩  ♩  ♩            ♩──♩  ♩──♩
  ▼  ▲  ▼  ▲            ▼───  ▲───
  (separate bows)        (2 notes per bow)
```

**How:** Keep the bow moving in the same direction while changing fingers.

### 2. String Crossings

```
  Smooth crossing:
  
  ──A string──┐
              └──E string──┐
                           └──A string──
  
  Keep your bow arm level change SMALL.
  Think: elbow drops slightly for lower strings,
         rises slightly for higher strings.
```

### 3. Dynamics

```
  pp   pianissimo    very soft      (whisper)
  p    piano         soft           (speaking quietly)
  mp   mezzo piano   medium soft    (normal talking)
  mf   mezzo forte   medium loud    (speaking up)
  f    forte         loud           (shouting)
  ff   fortissimo    very loud      (yelling)

  ────<────  crescendo (get louder)
  ────>────  decrescendo (get quieter)
```

### 4. The 4th Finger (Pinky)

```
  Your pinky is WEAK — that's normal. Practice this daily:

  On A string:
  open──[1]──[2]──[3]──[4]──[3]──[2]──[1]──open
  A      B    C#   D    E    D    C#   B    A
  A0     A1   A2   A3   A4   A3   A2   A1   A0

  Do this on EVERY string. Slowly. Pinky stays curved!
```

## Scales for This Level

### A Major (3 sharps: F#, C#, G#)
```
  Note:    A    B    C#   D    E    F#   G#   A
  Finger:  0    1    2    3    0    1    2    3
  String:  A    A    A    A    E    E    E    E
  Code:    A0   A1   A2   A3   E0   E1   E2   E3
  Pattern: hi2  hi2 (both strings same pattern)
```

### E Major (4 sharps: F#, C#, G#, D#)
```
  Note:    E    F#   G#   A    B    C#   D#   E
  Finger:  0    1    2    3    (higher positions needed)
  String:  E    E    E    E    
  Code:    E0   E1   E2   E3   (stay 1 octave for now)
```

### D Minor (1 flat: B♭)
```
  Note:    D    E    F    G    A    B♭   C#   D
  Finger:  0    1    2    3    0    1    2    3
  String:  D    D    D    D    A    A    A    A
  Code:    D0   D1   D2   D3   A0   A1   A2   A3
                   lo2              lo1  hi2
  
  Special: F = low 2nd finger on D string (D2 pulled back)
           B♭ = low 1st finger on A string (A1 pulled back)
```

### G Minor
```
  Note:    G    A    B♭   C    D    E♭   F#   G
  Finger:  0    1    2    3    0    1    2    3
  String:  G    G    G    G    D    D    D    D
  Code:    G0   G1   G2   G3   D0   D1   D2   D3
                    lo2              lo1  hi2
```

## Piece 7: Minuet No. 1 (Bach/Suzuki Book 1)

**Key:** G Major (F#) | **Time:** 3/4 (waltz — 3 beats per measure)

```
  Measure:  |1           |2           |3           |4           |
  Notes:     D    G    A | B    G    G | C    D    E | D    –    –
  Finger:    0    3    0 | 1    3    3 | 3    0    1 | 0
  String:    D    D    A | A    D    D | D    D    D | D
  Code:      D0   D3  A0| A1   D3   D3| D3   D0   D1| D0
  Count:     1    2    3 | 1    2    3 | 1    2    3 | 1    2    3

  Measure:  |5           |6           |7           |8           |
  Notes:     B    C    B | A    G    A | B    C    B | A    –    –
  Finger:    1    2    1 | 0    3    0 | 1    2    1 | 0
  String:    A    A    A | A    D    A | A    A    A | A
  Code:      A1   A2  A1| A0   D3  A0| A1   A2   A1| A0
```

> Note: C here is C natural (low 2nd finger on A string) because
> we're in G Major. Code is still A2, just with finger pulled back.

## 🎯 Practice Order for Level 2

```
  Week 7-8:    Long Long Ago + Minuet 1
  Week 9-10:   Allegretto + Minuet 2
  Week 11-12:  Allegro + Gavotte
```

### Daily Routine at This Level (25-30 min)

```
  1. Scales (5 min)
     - D Major, G Major, A Major
     - Play with SLURS (2 notes per bow, then 4)
  
  2. 4th finger exercise (3 min)
     - 0-1-2-3-4-3-2-1-0 on each string
  
  3. String crossing exercise (3 min)
     - Open A, open E, open A, open E (smooth bowing)
  
  4. Current piece — slow practice (10 min)
     - Measure by measure
  
  5. Current piece — play through (5 min)
     - With a metronome if possible
```
