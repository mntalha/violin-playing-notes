## 4. One-Octave Scales (♩= 69)

### C Major (no sharps or flats)

```
  Note:    C    D    E    F    G    A    B    C
  Finger:  3    4    1    2    3    0    1    2(4)
  String:  G    G    D    D    D    A    A    A
  Code:    G3   G4   D1   D2   D3   A0   A1   A2

  Remember: LOW 2nd finger on D string (F natural, not F#)
            LOW 2nd finger on A string (C natural going up to final C)
  
  Your handwritten codes under the staff confirm this:
  G3 → D0 → D1 → D2 → D3 → A0 → A1 → A2 (and continuing up)
  Then A0 → G3 → D2 → D1 → D0 → G3 (coming back down)
```

### A Minor Harmonic (raised 7th: G#)

```
  Note:    A    B    C    D    E    F    G#   A
  Finger:  0    1    2    3    0    1    2    3
  String:  A    A    A    A    E    E    E    E
  Code:    A0   A1   A2   A3   E0   E1   E2   E3

  ⭕ You circled the # (sharp) sign before G — this is the 
  RAISED 7TH that makes it HARMONIC minor.
  
  G# = E2 (high 2nd finger on E string)
  Without the #, it would be G natural = E2 low (natural minor)
  
  The sharp sign is what gives harmonic minor its distinctive 
  "exotic" sound — that big step from F to G#.
```

### D Minor Harmonic (1♭: B♭, raised 7th: C#)

```
  Note:    D    E    F    G    A    B♭   C#   D
  Finger:  0    1    2    3    0    1    2    3
  String:  D    D    D    D    A    A    A    A
  Code:    D0   D1   D2   D3   A0   A1   A2   A3

  Key points:
  - F = LOW 2nd finger on D string (D2 pulled back)
  - B♭ = LOW 1st finger on A string (A1 pulled back)
  - C# = HIGH 2nd finger on A string (A2, raised 7th)
  - That jump from B♭ to C# = augmented 2nd (the "exotic" interval)
```

## 6. Double Stop Exercise for Tone Development (♩= 60)

**Key:** G Major (1#: F#) | **Time:** 4/4 | **Bowing:** Half notes (𝅗𝅥)

```
  Play TWO open strings at once. Bow touches both strings equally.

  Pair 1:  G + D    (strings 4 + 3)    Perfect 5th
  Pair 2:  D + A    (strings 3 + 2)    Perfect 5th
  Pair 3:  A + E    (strings 2 + 1)    Perfect 5th
  Pair 4:  G + D    (repeat)
  ...continue pattern...
  Final:   All four open strings together (chord)

  Focus on:
  ✅ Even pressure on BOTH strings
  ✅ Slow, steady bow (half notes at ♩= 60 = very slow!)
  ✅ Clean sound — no scratching or bouncing
  ✅ Listen for the pure 5th "ring" — it should sound hollow and clear
```

## 7. March in C Major — Wohlfahrt Op. 38, No. 40

**Level 1 Étude** | **Tempo:** March ♩= 116–120 | **Key:** C Major | **Time:** 4/4 (C)

### Your Circled Markings Explained:

```
  ⭕ mf (mezzo forte) — Start at medium loud. It's a march, 
     so it should sound confident but not blasting.

  ⭕ Finger "2" — 2nd finger placement at the start.
     In C Major on the G string, 2nd finger = B (high 2).

  ⭕ Down bow (П) — Both violins start with a down bow.
     Down bow on beat 1 gives a strong, natural accent.

  ⭕ Up bow (V) — Violin 2 has an up bow marked mid-phrase.
     This ensures the bowing pattern lines up with the beat.

  ⭕ f (forte) — At measure 5, volume increases to LOUD.
     The march gets stronger here.

  ⭕ p (piano) — At measure 9, drop to SOFT.
     Creates contrast — a quiet section before building back up.

  ⭕ cresc. (crescendo) — Gradually get louder from this point.
     Both violin parts have cresc. at the same spot (m.9-10).
     Build from p back toward f.
```

### Violin 1 — Key Notes:

```
  Measures 1-4 (mf):
  Half notes and quarter notes — mostly on G and D strings.
  Strong march rhythm: long-short patterns.
  
  Measures 5-8 (f):
  More running quarter notes and eighth notes.
  String crossings between D and A strings.
  
  Measures 9-12 (p → cresc.):
  Start soft, build up. Same melodic material.
  
  Measures 13+ (f):
  Return to loud. Strong ending.
```

### Violin 2 — Key Notes:

```
  Accompaniment part — dotted rhythms (♩. ♪ pattern).
  Supports Violin 1's melody.
  Same dynamic changes (mf → f → p → cresc. → f).
  Watch the up bow (V) marking — important for staying 
  in sync with Violin 1.
```

## 10. Mischievously — Christine Donkin (b. 1976)

**Tempo:** ♩= 116–120 | **Style:** Mischievously (playful!) | **Two violin parts**

```
  Your circled markings:
  
  ⭕ + (plus sign above notes) — LEFT-HAND PIZZICATO
     This is an advanced technique! You pluck the string with 
     your LEFT hand finger instead of the bow.
     
     How to play +: 
     - Place finger on the note
     - Pull the finger sideways to pluck the string
     - Creates a short, percussive "ping" sound
     - Fits the "mischievous" character perfectly!
  
  ⭕ Quarter rest (𝄽) — silence for one beat
     Important in this piece — the rests create the 
     "sneaky" rhythmic feel.
  
  ⭕ f (forte) — sudden loud! 
     The piece plays with mf and sudden f bursts.
     This contrast IS the "mischievous" character.

  ⭕ Staccato dots (.) — play notes SHORT and bouncy.
     Combined with the + pizzicato markings, this piece 
     alternates between bowed staccato and plucked notes.
  
  Character: Think of a cat sneaking around, then pouncing!
  Both violin parts trade the melody back and forth.
```
