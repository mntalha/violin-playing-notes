## 🎯 The Code System

Every note uses a simple **string + finger** code so you always know exactly where to play:

```
G string:  G=G0   A=G1   B=G2   C=G3   D=G4
D string:  D=D0   E=D1   F#=D2  G=D3   A=D4
A string:  A=A0   B=A1   C#=A2  D=A3   E=A4
E string:  E=E0   F#=E1  G#=E2  A=E3   B=E4
```

`A1` = A string, 1st finger = **B**. That's it.

## 🕹️ Interactive Trainer App

Open **`violin-trainer.html`** in your browser (just double-click it). No internet required.

### 6 Practice Modes:

| Mode | What it does |
|------|-------------|
| **Code → Note** | See `A1`, pick the note name |
| **Note → Code** | See "B on A string", pick `A1` |
| **Staff → Code** | See a dot on the staff, pick the code |
| **✏️ Staff → Type** | See a dot on the staff, **type** the code from memory |
| **⚡ Staff Blitz** | Timed rounds that get faster — with combos, levels, and scoring |
| **Fingerboard Tap** | Tap the correct position on a visual fingerboard |

### Features:
- 🔊 **Hear each note** — built-in tone playback
- 🧠 **Learns from your mistakes** — wrong answers come back more often until you master them
- 🔥 **Streak & combo tracking** — stay motivated
- 🎯 **Filter by string** — focus on G, D, A, or E
- 💾 **Progress saved** — your trouble notes persist between sessions
- 📖 **Built-in reference card** — toggle it when you need help

## 🏆 Golden Rules

1. **Slow is fast** — practice slowly, speed comes naturally
2. **15 min daily > 2 hours weekly** — consistency wins
3. **Listen more than you play** — if it sounds wrong, stop and fix it
4. **Relax everything** — tension is the enemy of good violin playing
5. **Use the trainer daily** — even 5 minutes of note drilling builds fluency
