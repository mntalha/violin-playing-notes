# 🎵 Your First Notes — Fingers on the Strings

*You can hold the violin, hold the bow, and play open strings.*
*Now: place your left-hand fingers and play REAL notes!*

---

## 1. How Your Left Hand Works

You have 4 fingers to use (your thumb stays behind the neck):

```
  Index finger  =  Finger 1  (strongest, easiest)
  Middle finger =  Finger 2
  Ring finger   =  Finger 3
  Pinky         =  Finger 4  (weakest, hardest — be patient!)
```

When you press a finger down on a string, you **shorten** the vibrating
part of the string. Shorter string = higher pitch.

```
  Open string (no finger):     Lowest note on that string
  1st finger:                  One step higher
  2nd finger:                  Two steps higher
  3rd finger:                  Three steps higher
  4th finger:                  Four steps higher = same as next open string!
```

### How to Place Your Fingers

```
  ✅ Curved fingers — land on the TIPS, not flat pads
  ✅ Come from ABOVE the string, pressing straight down
  ✅ Press just hard enough to get a clear note (not harder!)
  ✅ Thumb stays BEHIND the neck, relaxed, opposite your 1st-2nd finger
  ✅ Keep fingers close to the string even when not pressing

  ❌ Don't flatten your fingers
  ❌ Don't squeeze the neck with your thumb
  ❌ Don't lift unused fingers high in the air
```

## 2. The Notes on Each String

Here's what happens when you put each finger on each string.
Don't worry about "sharps" or "naturals" yet — just learn where your
fingers go and what note comes out.

### 🔵 G String (String 4 — the thickest)

```
  Open    1st     2nd     3rd     4th
   G  ──  A  ──  B  ──   C  ──  D
   │      │      │       │      │
  no     index  middle  ring   pinky
  finger
```

### 🟢 D String (String 3)

```
  Open    1st     2nd     3rd     4th
   D  ──  E  ──  F# ──   G  ──  A
   │      │      │       │      │
  no     index  middle  ring   pinky
  finger
```

### 🟡 A String (String 2)

```
  Open    1st     2nd     3rd     4th
   A  ──  B  ──  C# ──   D  ──  E
   │      │      │       │      │
  no     index  middle  ring   pinky
  finger
```

### 🔴 E String (String 1 — the thinnest)

```
  Open    1st     2nd     3rd     4th
   E  ──  F# ──  G# ──   A  ──  B
   │      │      │       │      │
  no     index  middle  ring   pinky
  finger
```

> 💡 **Notice the pattern:** 4th finger on any string plays the SAME NOTE 
> as the NEXT open string!
> - 4th finger on G = **D** = open D string  
> - 4th finger on D = **A** = open A string
> - 4th finger on A = **E** = open E string

## 3. The Fingerboard — Full Picture

Looking down at your violin (scroll pointing up, strings left to right):

```
        G     D     A     E          ← Open strings (no fingers)
        │     │     │     │
  1st   A     E     B     F#         ← Index finger
        │     │     │     │
  2nd   B     F#    C#    G#         ← Middle finger (high position*)
        │     │     │     │
  3rd   C     G     D     A          ← Ring finger
        │     │     │     │
  4th   D     A     E     B          ← Pinky
```

> *This shows the "high 2nd finger" position — the default starting pattern.
> Your 2nd finger sits close to your 3rd finger. This gives you sharps 
> (F#, C#, G#). Later, you'll learn to move it back for a different sound.

## 4. The Code System — Your Shorthand

To make reading easier, we use a simple code: **String + Finger number**

```
  A0 = A string, open (no finger)     = note A
  A1 = A string, 1st finger           = note B
  A2 = A string, 2nd finger           = note C# (high 2)
  A3 = A string, 3rd finger           = note D
  D3 = D string, 3rd finger           = note G
  E0 = E string, open                 = note E
```

### Complete Code Reference

```
  G string:  G=G0   A=G1   B=G2   C=G3   D=G4
  D string:  D=D0   E=D1   F#=D2  G=D3   A=D4
  A string:  A=A0   B=A1   C#=A2  D=A3   E=A4
  E string:  E=E0   F#=E1  G#=E2  A=E3   B=E4
```

## 5. Your First Exercises

### Exercise 1: One Finger at a Time (on A string)

```
  Play each note slowly. Bow one down-up per note.

  Open   1st    Open   1st    Open   1st    Open
   A      B      A      B      A      B      A
   A0     A1     A0     A1     A0     A1     A0

  Listen: does the 1st finger B sound clear? 
  If it buzzes, press a tiny bit harder.
```

### Exercise 2: Walking Up (on A string)

```
  Open   1st    2nd    3rd     (and back down)
   A      B     C#     D       C#     B      A
   A0     A1    A2     A3      A2     A1     A0

  Keep your fingers DOWN as you add the next one.
  Don't lift 1st finger when you place 2nd!
```

### Exercise 3: Walking Up (on D string)

```
  Open   1st    2nd    3rd     (and back down)
   D      E     F#     G       F#     E      D
   D0     D1    D2     D3      D2     D1     D0
```

### Exercise 4: Crossing Strings

```
  A string:  A  B  C# D   (A0 A1 A2 A3)
  E string:  E  F# G# A   (E0 E1 E2 E3)
  Back:      A  G# F# E   (E3 E2 E1 E0)
  Back:      D  C# B  A   (A3 A2 A1 A0)

  Practice the string crossing (A → E) slowly.
  Your bow arm moves slightly — elbow drops a little for E string.
```

## 6. Your First Songs!

### Twinkle Twinkle Little Star (A + E strings)

```
  A  A  E  E  F# F# E     D  D  C# C# B  B  A
  0  0  0  0  1  1  0     3  3  2  2  1  1  0
  A  A  E  E  E  E  E     A  A  A  A  A  A  A   ← string
  A0 A0 E0 E0 E1 E1 E0    A3 A3 A2 A2 A1 A1 A0  ← code
```

### Mary Had a Little Lamb (A + D strings)

```
  B  A  G  A  B  B  B     A  A  A     B  D  D
  1  0  3  0  1  1  1     0  0  0     1  3  3
  A  A  D  A  A  A  A     A  A  A     A  A  A
  A1 A0 D3 A0 A1 A1 A1    A0 A0 A0   A1 A3 A3
```

### Hot Cross Buns (A + D strings)

```
  B  A  G       B  A  G
  1  0  3       1  0  3
  A  A  D       A  A  D
  A1 A0 D3      A1 A0 D3

  G  G  G  G   A  A  A  A   B  A  G
  3  3  3  3   0  0  0  0   1  0  3
  D  D  D  D   A  A  A  A   A  A  D
  D3 D3 D3 D3  A0 A0 A0 A0  A1 A0 D3
```

## 7. Daily Practice at This Stage (15 min)

```
  1. Open strings — 3 minutes
     Long, slow bows on G, D, A, E
     Focus on clear, even tone

  2. Finger exercises — 5 minutes
     Walk up 0-1-2-3 and back on each string
     Keep fingers curved, listen for clean notes

  3. String crossing — 2 minutes
     Play open A → open E → open A → open E
     Smooth, no bumps or scratches at the crossing

  4. One song — 5 minutes
     Pick one: Hot Cross Buns, Mary Had a Little Lamb, or Twinkle
     Play SLOWLY — get every note right before speeding up
```
