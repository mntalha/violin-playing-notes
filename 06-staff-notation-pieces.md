# 🎼 Sheet Music — Notes on the Staff (5-Line Notation)

*Each piece is written on the actual 5-line staff so you can practice reading.*

**Code format:** `A1` = A string, 1st finger. `E0` = E string, open. `D3` = D string, 3rd finger.

---

## How to Read These Diagrams

```
  ● ON a line    = note sits directly on the line
  ● BETWEEN lines = note sits in the space between two lines
  
  Lines are labeled 1-5 (bottom to top)
  Spaces between lines are shown as blank rows
```

### The Staff Map (memorize this!)

```
                                        Code
  ═══5═══  F / F#                        E1
           E  ●                          E0
  ═══4═══  D                             A3
           C / C#  ●                     A2
  ═══3═══  B                             A1
           A  ●                          A0
  ═══2═══  G                             D3
           F / F#  ●                     D2
  ═══1═══  E                             D1
           D  ●    (below staff)         D0
  - - - -  C       (ledger)             G3
           B  ●                          G2
  - - - -  A       (ledger)             G1
           G  ●                          G0
```

## 🎯 How to Practice Reading

```
  Step 1:  Is the note ON a line or BETWEEN two lines?
  
  Step 2:  Which line or space?
           Lines:  1=E  2=G  3=B  4=D  5=F
           Spaces: 1=F  2=A  3=C  4=E
  
  Step 3:  Say the code
           e.g. "That's line 3... B... A1!"
  
  Step 4:  Play it — 1st finger on A string
  
  Step 5:  Does it sound right? Check against open string.

  PRACTICE DAILY. Start with 5 minutes of note naming.
```

## 🧪 Test Yourself!

Can you name these notes? Read from left to right:

```
  ═══5═══──────────────────────────────────────────
                                ●
  ═══4═══──────────────────────────────────────────
     ●                                   ●
  ═══3═══──────────────────────────────────────────
                    ●         
  ═══2═══──●───────────────────────────────────────
                                              ●
  ═══1═══──────────────────────────────────────────
```

**Answers (don't peek!):**

```
  1st ● = line 2     = G  = D3
  2nd ● = space 3    = C  = A2 (or C# in D Major)
  3rd ● = space 2    = A  = A0
  4th ● = space 4    = E  = E0
  5th ● = space 3    = C  = A2
  6th ● = space 1    = F  = D2 (or F# in D Major)
```
